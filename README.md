# Job Recommender

A deep learning-based job recommender system.

## Installation

- To install, run the following conda command

```
conda env create -f environment.yml
```

## How to Train

- To train the model, run the following python command:

```
python main.py
```

## Command Options

- Command options are available in lib/config.py

## Dataset

- If the dataset is not installed, it will be automatically installed for you.
